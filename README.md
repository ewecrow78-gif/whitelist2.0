## ⚙️ Как использовать

1. Скопируй ссылку нужной подписки из таблицы выше
2. Открой VPN-клиент → Добавить подписку → Вставь ссылку
3. Обнови подписку в клиенте
4. Подключайся к любому серверу из списка
